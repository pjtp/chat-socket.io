## Getting Started

### Prerequisites

- Node.js (v18.x or latest LTS recommended)
- npm or yarn

### Installation

1. Clone the repository:

```bash
git clone https://github.com/pjtp/chat-socket.io.git
cd chat-socket.io

# For frontend:
cd client
npm install

# For backend:
cd server
npm install

# For frontend:
cd client
npm start

# For backend:
cd server
node index.js
```
